COMP 299: Future Interactions

Project Broomstick is the development of a musical conducting game!
This game is for everyone of all ages who want to begin interacting with music in a new way!

Conducting is the art of directing a musical performance by visible gestures.
The conductor's job is to unify performers, set the tempo, cue, and shape the sound of the piece.

Our project is available with use of a Leap Motion, a sensor device requiring no hand contact or touching!
We chose this device because it made it more realistic in trying to create the gestures of a conductor.

We're currently open source! Visit our Github repository.
This repository is where we store all of our work, it is great because it allows collaboration on the project!
Each contributor can add or change the files on the repository and it remains up to date!
